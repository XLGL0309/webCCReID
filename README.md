# 换衣行人重识别Web检索系统

## 项目简介

这是一个基于Flask的换衣行人重识别Web检索系统，用户可以上传一张探针图（目标行人照片）和一个包含多张备选图的底库文件夹，系统会自动提取特征并计算相似度，返回最相似的匹配结果。

## 环境要求

- Python 3.7+
- 所需依赖包见requirements.txt

## 安装步骤

1. 克隆或下载项目到本地
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```
3. 修改模型路径：
   在app.py文件中，修改model_path变量为您本地训练好的模型路径

## 使用方法

1. 启动应用：
   ```bash
   python app.py
   ```
2. 在浏览器中访问：http://localhost:5000
3. 上传探针图和底库文件夹
4. 点击"开始检索"按钮
5. 查看检索结果

## 项目结构

```
webCCReID/
├── app.py              # 主应用文件
├── model.py            # 模型加载和特征提取
├── templates/
│   ├── index.html      # 上传页面
│   └── result.html     # 结果展示页面
├── uploads/            # 上传文件存储目录
└── requirements.txt    # 依赖包列表
```

## 注意事项

1. 模型结构：model.py中的Model类是一个示例结构，您需要根据实际的模型结构进行修改
2. 输入大小：默认输入大小为256x128，您可以根据模型要求进行调整
3. 特征维度：默认输出特征维度为2048，您需要根据实际模型输出进行调整
4. 文件类型：仅支持png、jpg、jpeg格式的图片
5. 上传限制：由于Web服务器的限制，上传文件大小可能有限制

## 故障排除

1. 如果出现模型加载错误，请检查模型路径是否正确
2. 如果出现特征提取错误，请检查模型结构是否与实际模型匹配
3. 如果出现文件上传错误，请检查文件类型是否正确
